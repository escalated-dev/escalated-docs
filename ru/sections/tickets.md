# Тикеты

Тикеты -- это ядро Escalated. Каждый тикет представляет собой долгоживущий конечный автомат с полной историей переписки, уровнями приоритета, отслеживанием назначений и мониторингом SLA.

> **Примечание:** Каждый тикет получает уникальный номер (например, `ESC-00001`) и поддерживает настраиваемые статусы, уровни приоритета, назначение отдела, теги, отслеживание SLA и насыщенные переписки с вложенными файлами. Все данные хранятся в вашей базе данных.

## Расширенные рабочие процессы тикетов

- Процесс объединения тикетов для слияния дубликатов с сохранением истории
- Связанные тикеты для рабочих процессов инцидентов/проблем
- Побочные разговоры для параллельной внутренней/внешней координации
- Поддержка лёгких агентов для ограниченных операционных ролей
- Формы настраиваемых полей с условными правилами на основе контекста тикета

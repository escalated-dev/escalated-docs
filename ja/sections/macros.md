# マクロ

マクロは、エージェントがワンクリックでチケットに適用できる再利用可能なアクションシーケンスです。1つのマクロでステータスの変更、優先度の設定、エージェントの割り当て、返信の追加などを一度に実行できます。

## マクロアクション

各マクロには順番に実行される1つ以上のアクションが含まれます：

- **ステータスの変更** -- チケットのステータスを設定
- **優先度の変更** -- チケットの優先度を設定
- **エージェントの割り当て** -- チケットを特定のエージェントに割り当て
- **部門の変更** -- チケットを部門に移動
- **タグの追加** -- チケットにタグを適用
- **返信の送信** -- 会話に公開返信を追加
- **メモの追加** -- エージェントのみに表示される内部メモを追加

## 共有マクロとパーソナルマクロ

- **共有マクロ**はすべてのエージェントに表示され、管理者が管理します
- **パーソナルマクロ**は作成したエージェントにのみ表示されます

管理者は`Admin -> Macros`ページからマクロを管理できます。アクションビルダーで各ステップの追加、並べ替え、設定を視覚的に行えます。

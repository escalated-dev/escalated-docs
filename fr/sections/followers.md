# Abonnes de tickets

Les agents peuvent suivre des tickets pour recevoir des notifications sur les mises a jour sans etre l'assignataire. C'est utile pour les responsables qui souhaitent une supervision, ou pour les agents qui ont escalade un ticket mais veulent rester informes.

- Bouton suivre/ne plus suivre en un clic sur la page de detail du ticket
- Nombre d'abonnes affiche sur le bouton de suivi
- Les abonnes sont notifies lors de nouvelles reponses et de changements de statut
- Puce de filtre rapide "Following" pour trouver les tickets que vous surveillez
- Filtrer par "Following" dans les filtres avances de tickets

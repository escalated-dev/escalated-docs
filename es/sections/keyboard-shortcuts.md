# Atajos de teclado

Navega y actua sobre los tickets sin necesidad del raton. Los atajos son sensibles al contexto y se desactivan automaticamente cuando escribes en campos de entrada.

## Lista de tickets

| Tecla | Accion |
| --- | --- |
| `j` | Mover seleccion hacia abajo |
| `k` | Mover seleccion hacia arriba |
| `x` | Alternar seleccion de la fila actual |
| `Enter` | Abrir ticket seleccionado |

## Detalle del ticket

| Tecla | Accion |
| --- | --- |
| `r` | Enfocar el editor de respuesta |
| `n` | Cambiar a nota interna |
| `s` | Cambiar estado |
| `p` | Cambiar prioridad |
| `f` | Seguir / dejar de seguir ticket |

## Global

| Tecla | Accion |
| --- | --- |
| `?` | Mostrar panel de ayuda de atajos de teclado |

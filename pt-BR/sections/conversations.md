# Conversas

Cada ticket contem uma conversa encadeada entre o solicitante e os agentes de suporte. Notas internas permitem que sua equipe colabore sem expor mensagens ao cliente.

> **Nota:**
>
> - Respostas publicas visiveis tanto para o cliente quanto para os agentes
> - Notas internas visiveis apenas para os agentes
> - Anexos de arquivos nas respostas (limites configuraveis de tamanho e quantidade)
> - Respostas prontas para respostas comuns dos agentes
> - Notas internas fixaveis -- fixe notas importantes no topo do ticket para referencia rapida
> - Conversas paralelas para colaboradores fora da thread principal do solicitante

## 1. Installer le package

```bash
$ npm install @escalated-dev/escalated-adonis
```

## 2. Configurer le package

```bash
$ node ace configure @escalated-dev/escalated-adonis
```

## 3. Executer les migrations

```bash
$ node ace migration:run
```

> **Remarque :** La commande de configuration publie `config/escalated.ts`, enregistre le provider et copie les migrations automatiquement.

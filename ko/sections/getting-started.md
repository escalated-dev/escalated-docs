# 시작하기

Escalated는 공유 Inertia.js UI를 갖춘 오픈소스 임베디드 지원 티켓 시스템입니다. 프레임워크에 맞는 백엔드 어댑터를 설치하고, 프론트엔드용 npm 패키지를 추가하면 바로 사용할 수 있습니다.

## 핵심 기능

- 전체 티켓 라이프사이클 관리: 생성, 할당, 답변, 해결, 종료, 재오픈
- SLA 정책 및 에스컬레이션 규칙(영업시간 지원 포함)
- 에이전트 도구: 일괄 작업, 매크로, 팔로워, 고정 메모, 충돌 감지, 입력 표시기
- 협업 워크플로우: 사이드 대화, 티켓 연결, 티켓 병합, 라이트 에이전트 접근
- 관리 모듈: 커스텀 필드, 상태, 영업시간, 역할, 스킬, 용량, 자동화, Webhook
- 고객 경험: 포털, 게스트 티켓, 만족도 평가, 지식 베이스

## 고급 관리 기능

- 보고서 모음: 대시보드, 에이전트 생산성, SLA, CSAT 보고서
- CSAT 설문 설정 및 상세 평가 분석
- 보안 관리: SSO 설정 및 이중 인증 설정
- 데이터 보존 설정 및 예약 퍼지 도구
- 설정 가능한 스키마와 레코드를 갖춘 커스텀 객체

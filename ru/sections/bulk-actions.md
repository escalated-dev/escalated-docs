# Массовые действия

Выберите несколько тикетов из списка и примените действия пакетно. Агенты и администраторы могут изменять статус, приоритет, назначение, отдел, теги или удалять тикеты за одну операцию.

> **Доступные массовые действия**
>
> - **Изменение статуса** -- переведите все выбранные тикеты в новый статус
> - **Изменение приоритета** -- установите уровень приоритета для выбранных тикетов
> - **Назначение агента** -- назначьте или переназначьте агента на все выбранные тикеты
> - **Изменение отдела** -- переместите тикеты в другой отдел
> - **Добавление тегов** -- примените теги ко всем выбранным тикетам
> - **Удаление** -- безвозвратно удалите выбранные тикеты

## Быстрые фильтры

Список тикетов включает чипы быстрой фильтрации для типичных представлений. Один клик применяет предустановленный фильтр.

- Мои тикеты
- Неназначенные
- Urgent+
- SLA Breaching
- Отслеживаемые

## Улучшенный список тикетов

Список тикетов теперь содержит более подробную информацию с первого взгляда:

- Чекбоксы для выбора с опцией «выбрать все» для массовых операций
- Колонка с именем отправителя
- Время последнего ответа с относительными метками (например, «2ч назад»)
- Точечный индикатор статуса SLA (зелёный, жёлтый, красный)

# Primeros pasos

Escalated es un sistema de tickets de soporte de codigo abierto, integrable, con una interfaz compartida de Inertia.js. Instala el adaptador del backend para tu framework, agrega el paquete npm para el frontend, y estaras listo.

## Funcionalidades principales

- Ciclo de vida completo de tickets: crear, asignar, responder, resolver, cerrar, reabrir
- Politicas de SLA y reglas de escalamiento con soporte para horarios laborales
- Herramientas para agentes: acciones masivas, macros, seguidores, notas fijadas, deteccion de colisiones, indicadores de escritura
- Flujos de colaboracion: conversaciones paralelas, vinculacion de tickets, fusion de tickets, acceso de agente limitado
- Modulos de administracion: campos personalizados, estados, horarios laborales, roles, habilidades, capacidad, automatizaciones, webhooks
- Experiencia del cliente: portal, tickets de invitados, valoraciones de satisfaccion y base de conocimiento

## Funciones avanzadas de administracion

- Suite de informes: panel de control, productividad de agentes, SLA e informes CSAT
- Configuracion de encuestas CSAT y analiticas detalladas de valoraciones
- Controles de seguridad: configuracion de SSO y autenticacion de dos factores
- Configuracion de retencion de datos con herramientas de purga programada
- Objetos personalizados con esquemas y registros configurables

# 키보드 단축키

마우스 없이 티켓을 탐색하고 작업할 수 있습니다. 단축키는 컨텍스트를 인식하며, 입력 필드에서 타이핑 중일 때는 자동으로 비활성화됩니다.

## 티켓 목록

| 키 | 동작 |
| --- | --- |
| `j` | 선택을 아래로 이동 |
| `k` | 선택을 위로 이동 |
| `x` | 현재 행 선택 전환 |
| `Enter` | 선택한 티켓 열기 |

## 티켓 상세

| 키 | 동작 |
| --- | --- |
| `r` | 답변 작성기에 포커스 |
| `n` | 내부 메모로 전환 |
| `s` | 상태 변경 |
| `p` | 우선순위 변경 |
| `f` | 티켓 팔로우/언팔로우 |

## 전역

| 키 | 동작 |
| --- | --- |
| `?` | 키보드 단축키 도움말 오버레이 표시 |

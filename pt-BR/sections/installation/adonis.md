## 1. Instale o pacote

```bash
$ npm install @escalated-dev/escalated-adonis
```

## 2. Configure o pacote

```bash
$ node ace configure @escalated-dev/escalated-adonis
```

## 3. Execute as migracoes

```bash
$ node ace migration:run
```

> **Nota:** O comando configure publica `config/escalated.ts`, registra o provider e copia as migracoes automaticamente.

## 1. Zainstaluj pakiet

```bash
$ npm install @escalated-dev/escalated-adonis
```

## 2. Skonfiguruj pakiet

```bash
$ node ace configure @escalated-dev/escalated-adonis
```

## 3. Uruchom migracje

```bash
$ node ace migration:run
```

> **Uwaga:** Komenda configure publikuje `config/escalated.ts`, rejestruje providera i automatycznie kopiuje migracje.

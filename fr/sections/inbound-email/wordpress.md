### Configuration

Configurez l'e-mail entrant dans **Escalated -> Settings -> Email** dans wp-admin. Prend en charge les webhooks Mailgun, Postmark et AWS SES.

### Sondage IMAP

Le sondage IMAP s'execute via WP-Cron. Le plugin le planifie automatiquement lorsque IMAP est configure comme adaptateur d'e-mail entrant.

# اختصارات لوحة المفاتيح

تنقل وتصرف في التذاكر دون الحاجة للفأرة. الاختصارات واعية بالسياق وتُعطّل تلقائيًا عند الكتابة في حقول الإدخال.

## قائمة التذاكر

| المفتاح | الإجراء |
| --- | --- |
| `j` | تحريك التحديد للأسفل |
| `k` | تحريك التحديد للأعلى |
| `x` | تبديل تحديد الصف الحالي |
| `Enter` | فتح التذكرة المحددة |

## تفاصيل التذكرة

| المفتاح | الإجراء |
| --- | --- |
| `r` | التركيز على محرر الرد |
| `n` | التبديل إلى الملاحظة الداخلية |
| `s` | تغيير الحالة |
| `p` | تغيير الأولوية |
| `f` | متابعة / إلغاء متابعة التذكرة |

## عام

| المفتاح | الإجراء |
| --- | --- |
| `?` | عرض نافذة مساعدة اختصارات لوحة المفاتيح |

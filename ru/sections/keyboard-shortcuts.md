# Горячие клавиши

Навигация и действия с тикетами без мыши. Горячие клавиши контекстно-зависимы и автоматически отключаются при вводе текста в поля ввода.

## Список тикетов

| Клавиша | Действие |
| --- | --- |
| `j` | Переместить выделение вниз |
| `k` | Переместить выделение вверх |
| `x` | Переключить выбор текущей строки |
| `Enter` | Открыть выбранный тикет |

## Детали тикета

| Клавиша | Действие |
| --- | --- |
| `r` | Фокус на редакторе ответа |
| `n` | Переключиться на внутреннюю заметку |
| `s` | Изменить статус |
| `p` | Изменить приоритет |
| `f` | Подписаться / отписаться от тикета |

## Глобальные

| Клавиша | Действие |
| --- | --- |
| `?` | Показать справку по горячим клавишам |

## 1. Instalar el paquete

```bash
$ npm install @escalated-dev/escalated-adonis
```

## 2. Configurar el paquete

```bash
$ node ace configure @escalated-dev/escalated-adonis
```

## 3. Ejecutar migraciones

```bash
$ node ace migration:run
```

> **Nota:** El comando de configuracion publica `config/escalated.ts`, registra el provider y copia las migraciones automaticamente.

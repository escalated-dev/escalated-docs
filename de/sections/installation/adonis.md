## 1. Paket installieren

```bash
$ npm install @escalated-dev/escalated-adonis
```

## 2. Paket konfigurieren

```bash
$ node ace configure @escalated-dev/escalated-adonis
```

## 3. Migrationen ausführen

```bash
$ node ace migration:run
```

> **Hinweis:** Der configure-Befehl veröffentlicht `config/escalated.ts`, registriert den Provider und kopiert Migrationen automatisch.

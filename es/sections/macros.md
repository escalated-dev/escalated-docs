# Macros

Las macros son secuencias de acciones reutilizables que los agentes pueden aplicar a un ticket con un solo clic. Una unica macro puede cambiar el estado, establecer la prioridad, asignar un agente, agregar una respuesta y mas, todo a la vez.

## Acciones de macros

Cada macro contiene una o mas acciones que se ejecutan en orden:

- **Cambiar estado** -- establecer el estado del ticket
- **Cambiar prioridad** -- establecer la prioridad del ticket
- **Asignar agente** -- asignar el ticket a un agente especifico
- **Cambiar departamento** -- mover el ticket a un departamento
- **Agregar etiquetas** -- aplicar etiquetas al ticket
- **Enviar respuesta** -- agregar una respuesta publica a la conversacion
- **Agregar nota** -- agregar una nota interna visible solo para los agentes

## Macros compartidas vs. personales

- Las **macros compartidas** son visibles para todos los agentes y gestionadas por administradores
- Las **macros personales** son visibles solo para el agente que las creo

Los administradores gestionan las macros desde la pagina `Admin -> Macros`. El constructor de acciones permite agregar, reordenar y configurar cada paso de forma visual.

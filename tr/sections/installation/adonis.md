## 1. Paketi kurun

```bash
$ npm install @escalated-dev/escalated-adonis
```

## 2. Paketi yapılandırın

```bash
$ node ace configure @escalated-dev/escalated-adonis
```

## 3. Migration'ları çalıştırın

```bash
$ node ace migration:run
```

> **Not:** Yapılandırma komutu `config/escalated.ts` dosyasını yayınlar, sağlayıcıyı kaydeder ve migration'ları otomatik olarak kopyalar.

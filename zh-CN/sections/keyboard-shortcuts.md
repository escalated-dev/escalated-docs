# 键盘快捷键

无需鼠标即可导航和操作工单。快捷键具有上下文感知能力，在输入字段中打字时会自动禁用。

## 工单列表

| 按键 | 操作 |
| --- | --- |
| `j` | 向下移动选择 |
| `k` | 向上移动选择 |
| `x` | 切换当前行选择 |
| `Enter` | 打开选中的工单 |

## 工单详情

| 按键 | 操作 |
| --- | --- |
| `r` | 聚焦回复编辑器 |
| `n` | 切换到内部备注 |
| `s` | 更改状态 |
| `p` | 更改优先级 |
| `f` | 关注/取消关注工单 |

## 全局

| 按键 | 操作 |
| --- | --- |
| `?` | 显示键盘快捷键帮助覆盖层 |

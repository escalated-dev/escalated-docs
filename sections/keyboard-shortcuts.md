# Keyboard Shortcuts

Navigate and act on tickets without reaching for the mouse. Shortcuts are context-aware and automatically disabled when typing in input fields.

## Ticket List

| Key | Action |
| --- | --- |
| `j` | Move selection down |
| `k` | Move selection up |
| `x` | Toggle select current row |
| `Enter` | Open selected ticket |

## Ticket Detail

| Key | Action |
| --- | --- |
| `r` | Focus reply composer |
| `n` | Switch to internal note |
| `s` | Change status |
| `p` | Change priority |
| `f` | Follow / unfollow ticket |

## Global

| Key | Action |
| --- | --- |
| `?` | Show keyboard shortcut help overlay |

# Skróty klawiaturowe

Nawiguj i wykonuj akcje na zgłoszeniach bez sięgania po myszkę. Skróty są kontekstowe i automatycznie wyłączane podczas wpisywania tekstu w polach formularzy.

## Lista zgłoszeń

| Klawisz | Akcja |
| --- | --- |
| `j` | Przesuń zaznaczenie w dół |
| `k` | Przesuń zaznaczenie w górę |
| `x` | Przełącz zaznaczenie bieżącego wiersza |
| `Enter` | Otwórz zaznaczone zgłoszenie |

## Szczegóły zgłoszenia

| Klawisz | Akcja |
| --- | --- |
| `r` | Ustaw fokus na edytorze odpowiedzi |
| `n` | Przełącz na notatkę wewnętrzną |
| `s` | Zmień status |
| `p` | Zmień priorytet |
| `f` | Obserwuj / przestań obserwować zgłoszenie |

## Globalne

| Klawisz | Akcja |
| --- | --- |
| `?` | Pokaż nakładkę pomocy skrótów klawiaturowych |

# Makros

Makros sind wiederverwendbare Aktionssequenzen, die Agenten mit einem Klick auf ein Ticket anwenden können. Ein einzelnes Makro kann den Status ändern, die Priorität setzen, einen Agenten zuweisen, eine Antwort hinzufügen und mehr -- alles auf einmal.

## Makro-Aktionen

Jedes Makro enthält eine oder mehrere Aktionen, die der Reihe nach ausgeführt werden:

- **Status ändern** -- den Ticket-Status setzen
- **Priorität ändern** -- die Ticket-Priorität setzen
- **Agent zuweisen** -- das Ticket einem bestimmten Agenten zuweisen
- **Abteilung ändern** -- das Ticket in eine Abteilung verschieben
- **Tags hinzufügen** -- Tags auf das Ticket anwenden
- **Antwort senden** -- eine öffentliche Antwort zur Konversation hinzufügen
- **Notiz hinzufügen** -- eine interne Notiz hinzufügen, die nur für Agenten sichtbar ist

## Gemeinsame vs. persönliche Makros

- **Gemeinsame Makros** sind für alle Agenten sichtbar und werden von Admins verwaltet
- **Persönliche Makros** sind nur für den Agenten sichtbar, der sie erstellt hat

Admins verwalten Makros über die Seite `Admin -> Macros`. Der Aktions-Builder ermöglicht es, jeden Schritt visuell hinzuzufügen, neu zu ordnen und zu konfigurieren.

## 1. Установите пакет

```bash
$ npm install @escalated-dev/escalated-adonis
```

## 2. Настройте пакет

```bash
$ node ace configure @escalated-dev/escalated-adonis
```

## 3. Запустите миграции

```bash
$ node ace migration:run
```

> **Примечание:** Команда настройки публикует `config/escalated.ts`, регистрирует провайдер и копирует миграции автоматически.

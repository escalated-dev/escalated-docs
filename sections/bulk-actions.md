# Bulk Actions

Select multiple tickets from the list and apply actions in batch. Agents and admins can change status, priority, assignment, department, tags, or delete tickets in one operation.

> **Available bulk actions**
>
> - **Change status** -- move all selected tickets to a new status
> - **Change priority** -- set priority level across selected tickets
> - **Assign agent** -- assign or reassign an agent to all selected tickets
> - **Change department** -- move tickets to a different department
> - **Add tags** -- apply tags to all selected tickets
> - **Delete** -- permanently remove selected tickets

## Quick Filters

The ticket list includes quick filter chips for common views. One click applies a preset filter.

- My Tickets
- Unassigned
- Urgent+
- SLA Breaching
- Following

## Enhanced Ticket List

The ticket list now includes richer information at a glance:

- Checkbox selection with select-all for bulk operations
- Requester name column
- Last reply time with relative timestamps (e.g., "2h ago")
- SLA status dot indicator (green, yellow, red)

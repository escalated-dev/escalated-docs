# Pierwsze kroki

Escalated to otwartoźródłowy, osadzalny system zgłoszeń wsparcia ze współdzielonym interfejsem Inertia.js. Zainstaluj adapter backendowy dla swojego frameworka, dodaj pakiet npm dla frontendu i gotowe.

## Główne funkcjonalności

- Pełny cykl życia zgłoszenia: tworzenie, przypisywanie, odpowiadanie, rozwiązywanie, zamykanie, ponowne otwieranie
- Polityki SLA i reguły eskalacji z obsługą godzin pracy
- Narzędzia agentów: akcje zbiorcze, makra, obserwujący, przypięte notatki, wykrywanie kolizji, wskaźniki pisania
- Przepływy współpracy: rozmowy poboczne, łączenie zgłoszeń, scalanie zgłoszeń, dostęp jako light-agent
- Moduły administracyjne: pola niestandardowe, statusy, godziny pracy, role, umiejętności, pojemność, automatyzacje, webhooki
- Doświadczenie klienta: portal, zgłoszenia gości, oceny satysfakcji i baza wiedzy

## Zaawansowane funkcje administracyjne

- Pakiet raportów: panel główny, produktywność agentów, SLA i raporty CSAT
- Ustawienia ankiet CSAT i szczegółowa analityka ocen
- Kontrola bezpieczeństwa: ustawienia SSO i konfiguracja uwierzytelniania dwuskładnikowego
- Ustawienia retencji danych z narzędziami planowanego czyszczenia
- Obiekty niestandardowe z konfigurowalnymi schematami i rekordami

# Macros

Macros sao sequencias de acoes reutilizaveis que os agentes podem aplicar a um ticket com um unico clique. Uma macro pode alterar o status, definir a prioridade, atribuir um agente, adicionar uma resposta e mais -- tudo de uma vez.

## Acoes de macro

Cada macro contem uma ou mais acoes que sao executadas em ordem:

- **Alterar status** -- definir o status do ticket
- **Alterar prioridade** -- definir a prioridade do ticket
- **Atribuir agente** -- atribuir o ticket a um agente especifico
- **Alterar departamento** -- mover o ticket para um departamento
- **Adicionar tags** -- aplicar tags ao ticket
- **Enviar resposta** -- adicionar uma resposta publica a conversa
- **Adicionar nota** -- adicionar uma nota interna visivel apenas para agentes

## Macros compartilhadas vs. pessoais

- **Macros compartilhadas** sao visiveis para todos os agentes e gerenciadas pelos administradores
- **Macros pessoais** sao visiveis apenas para o agente que as criou

Administradores gerenciam macros na pagina `Admin -> Macros`. O construtor de acoes permite adicionar, reordenar e configurar cada etapa visualmente.

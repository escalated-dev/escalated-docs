# Macro's

Macro's zijn herbruikbare actiesequenties die agenten met een klik op een ticket kunnen toepassen. Een enkele macro kan de status wijzigen, de prioriteit instellen, een agent toewijzen, een antwoord toevoegen en meer -- alles in een keer.

## Macro-acties

Elke macro bevat een of meer acties die op volgorde worden uitgevoerd:

- **Status wijzigen** -- de ticketstatus instellen
- **Prioriteit wijzigen** -- de ticketprioriteit instellen
- **Agent toewijzen** -- het ticket aan een specifieke agent toewijzen
- **Afdeling wijzigen** -- het ticket naar een afdeling verplaatsen
- **Tags toevoegen** -- tags op het ticket toepassen
- **Antwoord versturen** -- een openbaar antwoord aan het gesprek toevoegen
- **Notitie toevoegen** -- een interne notitie toevoegen die alleen zichtbaar is voor agenten

## Gedeelde vs. persoonlijke macro's

- **Gedeelde macro's** zijn zichtbaar voor alle agenten en worden beheerd door beheerders
- **Persoonlijke macro's** zijn alleen zichtbaar voor de agent die ze heeft aangemaakt

Beheerders beheren macro's via de pagina `Admin -> Macros`. De actiebouwer laat je elke stap visueel toevoegen, herschikken en configureren.

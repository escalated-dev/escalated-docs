# Scorciatoie da Tastiera

Naviga e agisci sui ticket senza usare il mouse. Le scorciatoie sono sensibili al contesto e vengono disattivate automaticamente durante la digitazione nei campi di input.

## Lista Ticket

| Tasto | Azione |
| --- | --- |
| `j` | Sposta la selezione in basso |
| `k` | Sposta la selezione in alto |
| `x` | Seleziona/deseleziona la riga corrente |
| `Enter` | Apri il ticket selezionato |

## Dettaglio Ticket

| Tasto | Azione |
| --- | --- |
| `r` | Focus sul compositore di risposta |
| `n` | Passa alla nota interna |
| `s` | Cambia stato |
| `p` | Cambia priorità |
| `f` | Segui / non seguire il ticket |

## Globali

| Tasto | Azione |
| --- | --- |
| `?` | Mostra l'overlay di aiuto per le scorciatoie da tastiera |

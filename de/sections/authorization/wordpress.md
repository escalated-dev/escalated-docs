WordPress verwendet integrierte Rollen und Fähigkeiten. Das Plugin registriert zwei benutzerdefinierte Rollen:

| Rolle | Beschreibung |
| ----- | ------------ |
| `escalated_agent` | Kann Tickets anzeigen und verwalten |
| `escalated_admin` | Vollzugriff auf Einstellungen, Abteilungen und SLA-Richtlinien |

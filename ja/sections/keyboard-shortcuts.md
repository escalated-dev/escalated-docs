# キーボードショートカット

マウスに手を伸ばすことなく、チケットの操作やナビゲーションが可能です。ショートカットはコンテキストに応じて動作し、入力フィールドでの入力中は自動的に無効になります。

## チケット一覧

| キー | アクション |
| --- | --- |
| `j` | 選択を下に移動 |
| `k` | 選択を上に移動 |
| `x` | 現在の行の選択を切り替え |
| `Enter` | 選択したチケットを開く |

## チケット詳細

| キー | アクション |
| --- | --- |
| `r` | 返信コンポーザーにフォーカス |
| `n` | 内部メモに切り替え |
| `s` | ステータスを変更 |
| `p` | 優先度を変更 |
| `f` | チケットのフォロー/フォロー解除 |

## グローバル

| キー | アクション |
| --- | --- |
| `?` | キーボードショートカットのヘルプオーバーレイを表示 |

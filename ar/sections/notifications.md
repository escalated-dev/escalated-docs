# الإشعارات

يستخدم Escalated نظام الإشعارات الأصلي لإطار العمل الخاص بك. تُرسل إشعارات البريد الإلكتروني وقاعدة البيانات تلقائيًا للأحداث الرئيسية مثل التذاكر الجديدة، والردود، والتعيينات، وتغييرات الحالة، وانتهاكات SLA.

- يتم إشعار المعيّن للتذكرة بالردود الجديدة
- يتم إشعار مقدم الطلب بردود الوكلاء وتغييرات الحالة
- **متابعو التذاكر** يتلقون نفس الإشعارات التي يتلقاها المعيّن -- الردود وتغييرات الحالة
- تُرسل تنبيهات انتهاك SLA إلى المعيّن وأعضاء القسم

خصص قوالب البريد الإلكتروني بنشر العروض: `php artisan vendor:publish --tag=escalated-views`

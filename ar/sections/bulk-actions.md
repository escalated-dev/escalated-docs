# الإجراءات الجماعية

حدد عدة تذاكر من القائمة وطبق الإجراءات دفعة واحدة. يمكن للوكلاء والمسؤولين تغيير الحالة، والأولوية، والتعيين، والقسم، والوسوم، أو حذف التذاكر في عملية واحدة.

> **الإجراءات الجماعية المتاحة**
>
> - **تغيير الحالة** -- نقل جميع التذاكر المحددة إلى حالة جديدة
> - **تغيير الأولوية** -- تعيين مستوى الأولوية عبر التذاكر المحددة
> - **تعيين وكيل** -- تعيين أو إعادة تعيين وكيل لجميع التذاكر المحددة
> - **تغيير القسم** -- نقل التذاكر إلى قسم مختلف
> - **إضافة وسوم** -- تطبيق وسوم على جميع التذاكر المحددة
> - **حذف** -- إزالة التذاكر المحددة نهائيًا

## التصفية السريعة

تتضمن قائمة التذاكر شرائح تصفية سريعة للعروض الشائعة. نقرة واحدة تطبق تصفية محددة مسبقًا.

- تذاكري
- غير معيّنة
- Urgent+
- SLA Breaching
- المتابَعة

## قائمة التذاكر المحسّنة

تتضمن قائمة التذاكر الآن معلومات أغنى للعرض السريع:

- تحديد بخانة اختيار مع تحديد الكل للعمليات الجماعية
- عمود اسم مقدم الطلب
- وقت آخر رد مع طوابع زمنية نسبية (مثل "قبل ساعتين")
- مؤشر نقطي لحالة SLA (أخضر، أصفر، أحمر)

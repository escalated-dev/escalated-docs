# Raccourcis clavier

Naviguez et agissez sur les tickets sans toucher a la souris. Les raccourcis sont contextuels et automatiquement desactives lors de la saisie dans les champs de formulaire.

## Liste des tickets

| Touche | Action |
| --- | --- |
| `j` | Deplacer la selection vers le bas |
| `k` | Deplacer la selection vers le haut |
| `x` | Activer/desactiver la selection de la ligne courante |
| `Enter` | Ouvrir le ticket selectionne |

## Detail du ticket

| Touche | Action |
| --- | --- |
| `r` | Mettre le focus sur l'editeur de reponse |
| `n` | Basculer vers la note interne |
| `s` | Changer le statut |
| `p` | Changer la priorite |
| `f` | Suivre / ne plus suivre le ticket |

## Global

| Touche | Action |
| --- | --- |
| `?` | Afficher le panneau d'aide des raccourcis clavier |

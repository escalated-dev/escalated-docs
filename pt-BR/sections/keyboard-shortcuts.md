# Atalhos de Teclado

Navegue e atue nos tickets sem precisar usar o mouse. Os atalhos sao contextuais e desativados automaticamente ao digitar em campos de entrada.

## Lista de Tickets

| Tecla | Acao |
| --- | --- |
| `j` | Mover selecao para baixo |
| `k` | Mover selecao para cima |
| `x` | Alternar selecao da linha atual |
| `Enter` | Abrir ticket selecionado |

## Detalhe do Ticket

| Tecla | Acao |
| --- | --- |
| `r` | Focar no compositor de resposta |
| `n` | Alternar para nota interna |
| `s` | Alterar status |
| `p` | Alterar prioridade |
| `f` | Seguir / deixar de seguir ticket |

## Global

| Tecla | Acao |
| --- | --- |
| `?` | Mostrar sobreposicao de ajuda de atalhos de teclado |

# 에이전트 협업

실시간 프레젠스 표시기와 고정 내부 메모로 지원 팀의 동기화를 유지합니다. 외부 도구가 필요하지 않습니다.

## 충돌 감지

여러 에이전트가 동일한 티켓을 열었을 때, 각 에이전트에게 현재 누가 보고 있는지를 나타내는 아바타 표시기가 표시됩니다. 이를 통해 중복 답변과 불필요한 작업을 방지합니다.

- 다른 열람자의 이니셜이 표시되는 아바타 스택
- 호버 시 "Also viewing" 툴팁
- 30초마다 폴링 -- 가볍고 캐시 기반 (데이터베이스 테이블 불필요)

## 입력 표시기

에이전트는 답변 작성 중 활성 입력 상태를 브로드캐스트할 수 있어, 팀원이 중복 답변을 피할 수 있습니다.

- 티켓별 실시간 입력 상태
- 빠른 인수인계를 위한 충돌 감지 보완

## 고정 내부 메모

중요한 내부 메모를 티켓 상세 뷰 상단에 고정할 수 있습니다. 고정된 메모는 대화 스레드 위에 앰버 카드로 강조 표시되어, 에이전트가 중요한 컨텍스트를 놓치지 않습니다.

- 원클릭으로 내부 메모 고정/고정 해제
- 고정된 메모는 티켓 상단에 항상 표시
- 내부 메모만 고정 가능 -- 공개 답변은 고정 불가

## 지식 패널

티켓 사이드바에는 컨텍스트에 맞는 지식 패널이 포함되어 있어, 에이전트가 티켓 뷰를 벗어나지 않고 관련 도움말 콘텐츠를 참조할 수 있습니다.

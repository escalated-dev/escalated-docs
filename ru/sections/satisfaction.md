# Удовлетворённость клиентов

Собирайте обратную связь от клиентов после решения тикета. На странице тикета появляется простой запрос оценки по 5-звёздочной шкале с необязательным полем для комментария.

## Как это работает

- Когда тикет решён или закрыт, клиент видит запрос «Как вам наш сервис?»
- 5-звёздочная оценка с подписями при наведении (Ужасно, Плохо, Нормально, Хорошо, Отлично)
- Необязательный комментарий для подробной обратной связи
- Работает как для аутентифицированных клиентов, так и для владельцев гостевых тикетов
- Одна оценка на тикет -- при повторной отправке заменяется

## Для агентов и администраторов

- Оценка только для чтения отображается в боковой панели тикета
- Метрики CSAT на дашборде отчётов: средний балл, общее количество оценок и распределение по звёздам
- Специальная страница настроек CSAT для настройки поведения опроса и параметров сбора данных

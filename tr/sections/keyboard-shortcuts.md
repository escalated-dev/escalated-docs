# Klavye Kısayolları

Fareye uzanmadan biletlerde gezinin ve işlem yapın. Kısayollar bağlama duyarlıdır ve giriş alanlarında yazarken otomatik olarak devre dışı bırakılır.

## Bilet Listesi

| Tuş | Eylem |
| --- | --- |
| `j` | Seçimi aşağı taşı |
| `k` | Seçimi yukarı taşı |
| `x` | Geçerli satır seçimini aç/kapat |
| `Enter` | Seçili bileti aç |

## Bilet Detayı

| Tuş | Eylem |
| --- | --- |
| `r` | Yanıt düzenleyicisine odaklan |
| `n` | Dahili nota geç |
| `s` | Durumu değiştir |
| `p` | Önceliği değiştir |
| `f` | Bileti takip et / takibi bırak |

## Genel

| Tuş | Eylem |
| --- | --- |
| `?` | Klavye kısayolu yardım penceresini göster |

## 1. Installeer het pakket

```bash
$ npm install @escalated-dev/escalated-adonis
```

## 2. Configureer het pakket

```bash
$ node ace configure @escalated-dev/escalated-adonis
```

## 3. Voer migraties uit

```bash
$ node ace migration:run
```

> **Let op:** Het configure-commando publiceert `config/escalated.ts`, registreert de provider en kopieert migraties automatisch.

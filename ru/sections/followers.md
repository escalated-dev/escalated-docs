# Подписчики тикетов

Агенты могут подписаться на тикеты для получения уведомлений об обновлениях без назначения на тикет. Это полезно для менеджеров, желающих контролировать процесс, или агентов, которые эскалировали тикет, но хотят оставаться в курсе.

- Переключение подписки/отписки одним кликом на странице деталей тикета
- Количество подписчиков отображается на кнопке подписки
- Подписчики получают уведомления о новых ответах и изменениях статуса
- Чип быстрого фильтра «Отслеживаемые» для поиска тикетов, за которыми вы следите
- Фильтрация по «Отслеживаемые» в расширенных фильтрах тикетов

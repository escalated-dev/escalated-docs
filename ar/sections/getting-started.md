# البدء

Escalated هو نظام تذاكر دعم مفتوح المصدر وقابل للتضمين مع واجهة مستخدم مشتركة مبنية على Inertia.js. قم بتثبيت محول الواجهة الخلفية لإطار العمل الخاص بك، وأضف حزمة npm للواجهة الأمامية، وستكون جاهزًا للانطلاق.

## الإمكانيات الأساسية

- دورة حياة التذاكر الكاملة: إنشاء، تعيين، رد، حل، إغلاق، إعادة فتح
- سياسات SLA وقواعد التصعيد مع دعم ساعات العمل
- أدوات الوكلاء: إجراءات جماعية، ماكرو، متابعون، ملاحظات مثبتة، كشف التعارض، مؤشرات الكتابة
- سير عمل التعاون: محادثات جانبية، ربط التذاكر، دمج التذاكر، وصول الوكيل المحدود
- وحدات الإدارة: حقول مخصصة، حالات، ساعات العمل، أدوار، مهارات، سعة، أتمتة، خطافات الويب
- تجربة العملاء: بوابة، تذاكر الضيوف، تقييمات الرضا، وقاعدة المعرفة

## ميزات الإدارة المتقدمة

- مجموعة التقارير: لوحة المعلومات، إنتاجية الوكلاء، SLA، وتقارير CSAT
- إعدادات استبيان CSAT وتحليلات التقييم التفصيلية
- ضوابط الأمان: إعدادات SSO وإعداد المصادقة الثنائية
- إعدادات الاحتفاظ بالبيانات مع أدوات الحذف المجدولة
- كائنات مخصصة مع مخططات قابلة للتكوين وسجلات

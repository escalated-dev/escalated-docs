## 1. パッケージのインストール

```bash
$ npm install @escalated-dev/escalated-adonis
```

## 2. パッケージの設定

```bash
$ node ace configure @escalated-dev/escalated-adonis
```

## 3. マイグレーションの実行

```bash
$ node ace migration:run
```

> **注意：** configureコマンドは`config/escalated.ts`の公開、プロバイダーの登録、マイグレーションのコピーを自動的に行います。

# Sneltoetsen

Navigeer en handel tickets af zonder de muis te gebruiken. Sneltoetsen zijn contextbewust en worden automatisch uitgeschakeld bij het typen in invoervelden.

## Ticketlijst

| Toets | Actie |
| --- | --- |
| `j` | Selectie omlaag verplaatsen |
| `k` | Selectie omhoog verplaatsen |
| `x` | Huidige rij selecteren/deselecteren |
| `Enter` | Geselecteerd ticket openen |

## Ticketdetails

| Toets | Actie |
| --- | --- |
| `r` | Focus op antwoordveld |
| `n` | Wisselen naar interne notitie |
| `s` | Status wijzigen |
| `p` | Prioriteit wijzigen |
| `f` | Ticket volgen / niet meer volgen |

## Globaal

| Toets | Actie |
| --- | --- |
| `?` | Sneltoetshulp-overlay tonen |

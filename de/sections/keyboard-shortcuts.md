# Tastaturkürzel

Navigieren und handeln Sie bei Tickets, ohne zur Maus greifen zu müssen. Kürzel sind kontextabhängig und werden automatisch deaktiviert, wenn Sie in Eingabefeldern tippen.

## Ticketliste

| Taste | Aktion |
| ----- | ------ |
| `j` | Auswahl nach unten bewegen |
| `k` | Auswahl nach oben bewegen |
| `x` | Aktuelle Zeile auswählen/abwählen |
| `Enter` | Ausgewähltes Ticket öffnen |

## Ticket-Detailansicht

| Taste | Aktion |
| ----- | ------ |
| `r` | Antwort-Editor fokussieren |
| `n` | Zu interner Notiz wechseln |
| `s` | Status ändern |
| `p` | Priorität ändern |
| `f` | Ticket folgen / entfolgen |

## Global

| Taste | Aktion |
| ----- | ------ |
| `?` | Hilfe-Overlay für Tastaturkürzel anzeigen |

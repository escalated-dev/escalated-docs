# Pour commencer

Escalated est un systeme de tickets de support open source et integrable, avec une interface Inertia.js partagee. Installez l'adaptateur backend pour votre framework, ajoutez le package npm pour le frontend, et vous etes pret.

## Fonctionnalites principales

- Cycle de vie complet des tickets : creer, assigner, repondre, resoudre, cloturer, rouvrir
- Politiques SLA et regles d'escalade avec prise en charge des heures ouvrables
- Outils pour agents : actions groupees, macros, abonnes, notes epinglees, detection de collisions, indicateurs de saisie
- Flux de collaboration : conversations paralleles, liaison de tickets, fusion de tickets, acces agent limite
- Modules d'administration : champs personnalises, statuts, heures ouvrables, roles, competences, capacite, automatisations, webhooks
- Experience client : portail, tickets invites, evaluations de satisfaction et base de connaissances

## Fonctionnalites avancees d'administration

- Suite de rapports : tableau de bord, productivite des agents, SLA et rapports CSAT
- Parametres d'enquete CSAT et analyses detaillees des evaluations
- Controles de securite : parametres SSO et configuration de l'authentification a deux facteurs
- Parametres de retention des donnees avec outils de purge planifiee
- Objets personnalises avec schemas et enregistrements configurables

## 1. Installare il pacchetto

```bash
$ npm install @escalated-dev/escalated-adonis
```

## 2. Configurare il pacchetto

```bash
$ node ace configure @escalated-dev/escalated-adonis
```

## 3. Eseguire le migrazioni

```bash
$ node ace migration:run
```

> **Nota:** Il comando configure pubblica `config/escalated.ts`, registra il provider e copia le migrazioni automaticamente.

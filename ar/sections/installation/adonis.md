## 1. تثبيت الحزمة

```bash
$ npm install @escalated-dev/escalated-adonis
```

## 2. تكوين الحزمة

```bash
$ node ace configure @escalated-dev/escalated-adonis
```

## 3. تشغيل عمليات الترحيل

```bash
$ node ace migration:run
```

> **ملاحظة:** ينشر أمر التكوين `config/escalated.ts`، ويسجل المزود، وينسخ عمليات الترحيل تلقائيًا.

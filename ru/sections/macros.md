# Макросы

Макросы -- это многоразовые последовательности действий, которые агенты могут применить к тикету одним кликом. Один макрос может изменить статус, установить приоритет, назначить агента, добавить ответ и многое другое -- всё за один раз.

## Действия макросов

Каждый макрос содержит одно или несколько действий, выполняемых по порядку:

- **Изменение статуса** -- установка статуса тикета
- **Изменение приоритета** -- установка приоритета тикета
- **Назначение агента** -- назначение тикета конкретному агенту
- **Изменение отдела** -- перемещение тикета в отдел
- **Добавление тегов** -- применение тегов к тикету
- **Отправка ответа** -- добавление публичного ответа в переписку
- **Добавление заметки** -- добавление внутренней заметки, видимой только агентам

## Общие и личные макросы

- **Общие макросы** видны всем агентам и управляются администраторами
- **Личные макросы** видны только агенту, который их создал

Администраторы управляют макросами на странице `Admin -> Macros`. Конструктор действий позволяет добавлять, переупорядочивать и настраивать каждый шаг визуально.
